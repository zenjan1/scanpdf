🎉 ScanPDF v1.0.0 - 首次正式发布
================================================================================

📅 发布日期: 2026-07-06
📱 平台: Android
📦 文件大小: 43.5 MB

================================================================================
✨ 新功能
================================================================================

✅ 智能文档扫描
   - 相机实时预览
   - 自动边缘检测
   - 透视矫正
   - 图像增强（亮度、对比度）

✅ OCR 文字识别
   - 多语言支持（中文、英文、日文、韩文）
   - 基于 Google ML Kit
   - 实时文字提取

✅ PDF 生成
   - 多页文档合并
   - 自定义页面大小
   - 文字层嵌入（可搜索 PDF）

✅ 文档管理
   - 本地 SQLite 存储
   - 文档分类和标签
   - 快速搜索
   - 批量操作

✅ 云同步支持
   - 后端 API 支持
   - 文档上传/下载
   - 多设备同步

================================================================================
🔧 技术改进
================================================================================

• Flutter 3.24.0 + Dart 3.5.0
• Clean Architecture 架构
• BLoC 状态管理
• Google ML Kit OCR
• SQLite 本地存储
•  Material Design 3 UI

================================================================================
📱 安装说明
================================================================================

1. 下载 scanpdf-v1.0.0.apk
2. 传输到 Android 设备
3. 打开文件管理器，找到 APK 文件
4. 点击安装（需要启用"允许未知来源"）
5. 安装完成后打开应用

权限说明：
• 相机 - 拍摄文档
• 存储 - 保存扫描文档
• 网络 - 云同步（可选）

================================================================================
🐛 已知问题
================================================================================

• 相机预览在低版本 Android 上可能较慢
• OCR 识别大文件时需要较长处理时间
• 云同步功能需要自行部署后端
