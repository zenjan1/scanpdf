类似扫描全能王的专业文档扫描应用，支持自动边缘检测、透视矫正、OCR 文字识别、PDF 生成，多平台覆盖。

✨ 核心功能

专业级文档扫描，隐私优先，离线可用

📸

智能扫描

自动边缘检测、透视矫正，拍摄文档自动裁剪为平整的扫描件，效果媲美专业扫描仪。

🔤

OCR 文字识别

支持中文、英文、日文、韩文等多语言文字识别，生成可搜索的 PDF 文档。

🎨

图像增强

多种滤镜：灰度、黑白、魔法增强，自动去阴影、提高对比度、锐化文字。

📑

PDF 生成

多页合并、自定义页面大小、可搜索 PDF（含 OCR 文字层）、高质量输出。

📁

文档管理

标签分类、快速搜索、收藏标记、本地存储 + 云端同步，数据尽在掌握。

🔒

隐私保护

本地优先架构，数据加密存储，无广告无追踪，完全掌控您的文档数据。

🛠️ 技术架构

📱 客户端

  • Flutter 3.10+ 跨平台框架
  • BLoC 状态管理
  • Google ML Kit OCR
  • OpenCV 图像处理
  • SQLite 本地数据库
  • Dio HTTP 客户端

🖥️ 服务端

  • FastAPI 高性能框架
  • PostgreSQL 主数据库
