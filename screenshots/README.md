# 截图说明

由于无法运行 Flutter 应用，请手动创建以下截图：

## 必需截图（3-8张）

### 1. 主页/扫描界面
- 显示相机预览和扫描按钮
- 展示自动边缘检测效果

### 2. OCR 识别结果
- 显示扫描的文档
- 显示识别出的文字

### 3. PDF 生成功能
- 显示 PDF 列表
- 显示合并/拆分选项

### 4. 文档管理
- 显示文档列表
- 显示分类和标签

### 5. 云同步设置
- 显示同步开关
- 显示同步状态

## 截图要求
- 尺寸：1080x1920 或 1080x2340
- 格式：PNG
- 不要包含状态栏
- 不要包含其他应用边框

## 快速方法
1. 在手机上安装 ScanPDF APK
2. 打开应用，进入各个功能页面
3. 使用手机截图功能（电源键+音量下键）
4. 传输到电脑，上传到应用市场

或使用 Android 模拟器：
```bash
cd /home/a/scanpdf/flutter_app
flutter emulators --create
flutter emulators --launch <emulator_id>
flutter run
```
