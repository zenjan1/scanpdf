## 🏪 应用市场注册指南

### 1. Google Play（全球）

**费用**: $25 一次性
**注册链接**: https://play.google.com/console/signup

**步骤**:
1. 使用 Google 账号登录
2. 填写开发者信息
   - 开发者名称：ScanPDF Team 或你的姓名
   - 电子邮件：admin@zenjan.store
   - 网站：https://jp.zenjan.store
   - 隐私政策：https://jp.zenjan.store/privacy
3. 支付 $25 注册费（信用卡）
4. 完成身份验证（身份证/护照）
5. 等待审核（1-2 天）

**提交应用需要**:
- APK 或 AAB 文件
- 商店描述（中英文）
- 应用截图（手机：至少 2 张，平板：至少 2 张）
- 功能图（1024x500）
- 应用图标（512x512）
- 内容分级问卷
- 隐私政策链接

## 📸 应用截图需求

### 通用要求
- 格式：PNG 或 JPEG
- 无透明度
- 不包含其他设备边框
- 清晰展示应用功能

### 尺寸要求

**Android**:
- 手机：16:9 或 9:16，至少 320px，最大 3840px
- 平板 7"：16:9 或 9:16
- 平板 10"：16:9 或 9:16

**iOS**:
- iPhone 6.5"：1284x2778 或 1242x2688
- iPhone 5.5"：1242x2208 或 2208x1242
- iPad 12.9"：2048x2732 或 2732x2048

**推荐截图内容**:
1. 主页/扫描界面
2. OCR 识别结果
3. PDF 生成功能
4. 文档管理界面
5. 云同步功能
6. 设置页面

## 🌐 隐私政策托管

### 方案 1: GitHub Pages（免费）
1. 创建 GitHub 仓库：scanpdf-privacy
2. 上传 docs/privacy-policy.html
3. 启用 GitHub Pages
4. 访问：https://yourusername.github.io/scanpdf-privacy/

### 方案 2: 购买域名（推荐）
1. 购买域名：zenjan.store（当前使用 jp.zenjan.store）
2. 托管到 GitHub Pages / Vercel / Netlify
3. 访问：https://jp.zenjan.store/privacy

### 方案 3: 使用免费托管
- Vercel: https://vercel.com
- Netlify: https://netlify.com
- Cloudflare Pages: https://pages.cloudflare.com
