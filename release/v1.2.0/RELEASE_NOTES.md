# ScanPDF v1.2.0 发布说明

## 🎉 新功能

### 后端 API
- ✅ 分页、排序、过滤功能
- ✅ 批量操作（批量删除、批量收藏）
- ✅ 回收站功能（软删除、恢复、清空）
- ✅ 标签管理 API
- ✅ Token 刷新和密码重置

- ✅ 结果缓存机制（避免重复识别）

- ✅ 任务状态查询 API
- ✅ 优化的图片预处理流程

- ✅ BLoC 架构升级（分页、批量操作）

- ✅ 离线队列功能增强

- ✅ 密码重置功能
